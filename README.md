# CSV家計簿分析アプリ



## 概要

CSVファイルを読み込み、支出データをグラフで可視化できる家計簿Webアプリです。

カテゴリ別の円グラフや年別の推移グラフにより、支出の傾向を直感的に把握できます。

スマートフォンにも対応しており、場所を問わず支出の確認・管理が可能です。



## 作成背景

日々の支出を数値だけで管理するのは把握しづらいと感じ、

視覚的に分かりやすく分析できるアプリを作成しました。

また、日常的に使いやすいようスマートフォン対応も意識しました。



## 主な機能

・CSVファイルの読み込み

・月ごとの支出データ表示（商品名・数量・金額・カテゴリ）

・カテゴリ別の円グラフ表示

・年間支出の折れ線グラフ表示（年ごとに色分け）

・表示する年の切り替え機能

・固定費の入力・編集機能

・合計金額の自動計算

・データの自動保存機能（再読み込み時も保持）

・スマートフォン対応（レスポンシブデザイン）

## 工夫した点

・年ごとにグラフを切り替えられるようにし、複数年の比較をしやすくした

・固定費を後から編集できるようにして、実用性を高めた

・データを自動保存することで、継続的に使えるようにした

・スマートフォンでも操作しやすいUI設計を意識した

・CSV読み込みだけでなく、テキスト入力にも対応することで入力方法の柔軟性を向上させた

・スマートフォンでの操作性を考慮し、入力の手間を減らす設計にした



## 解決した課題

・支出データを数値だけで管理すると把握しづらいという問題を、グラフによる可視化によって直感的に理解できるようにした。

## 今後の改善点

・グラフのカスタマイズ機能の追加

・データのエクスポート機能(機能追加済み）



## 使い方

1. CSVファイルを読み込む

2. 月ごとの支出やグラフを確認

3. 必要に応じて固定費を入力・編集
